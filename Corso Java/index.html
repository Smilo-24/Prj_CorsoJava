Diventare Full Stack Developer

  • Java
  • JavaScript

Mission del Corso

Lo scopo e l'obiettivo del presente corso consiste nel fornire una preparazione completa per diventare un programmatore Full-Stack specializzato in Java. Un ulteriore obiettivo del corso è quello di sviluppare le competenze tecniche necessarie e le capacità comportamentali essenziali per interagire efficacemente con i clienti. Sarà insegnato come collaborare in modo efficace con i colleghi al fine di implementare un modello di lavoro di squadra consolidato all'interno dell'azienda. L'obiettivo finale del corso è quello di facilitare l'inserimento nell'ambiente lavorativo aziendale, seguendo i seguenti step:

Back-End

  • Conoscenza base di linguaggi quali HTML e CSS
  • Doti di programmazione e progettazione dei software
  • Dimestichezza con i framework come Spring, JPA, Hibernate
  • Familiarità con i Web Service più comuni - SOAP/REST
  • Conoscenza delle principali funzioni dei Database - MySQL
  • Sviluppo e studio di Java

Front-End

  • HTML, usato per dare una struttura agli elementi della pagina
  • CSS,usato per formattare gli elementi
  • JavaScript, usato per dare dinamicità agli elementi
  • Librerie e framework - Angular
  • Processori CSS(SASS e LESS sono i più richiesti)
  • Angular

Attraverso completamento di questo corso, sarò in grado di affrontare con successo le sfide professionali nel settore della programmazione Full-Stack, dimostrando sia le competenze tecniche necessarie che le abilità relazionali richieste per un efficace coinvolgimento con i clienti e il team di lavoro..
